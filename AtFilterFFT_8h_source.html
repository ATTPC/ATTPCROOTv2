1 /*
2  * Filter for applying a FFT to data, and multiplying the transformed data by some factor. The factor
3  * is determined by a series of FreqRanges that define a start and stop frequency and associated
4  * factor at the start/stop. It then linearly interpelates the factor between the two frequencies.
5  * By default the factor is 1 (i.e. no adjustment to the transformed data). For alternative behavior,
6  * extend the class and override the protected function applyFreqCuts().
7  *
8  * The output of this filter is an AtRawEvent where the fPadList is filled with pads of the type
9  * AtPadFFT so the fourier-space representation of the filtered data is saved.
10  * If you wish to save the unfiltered data with the fourier-space represetation, the set the flag
11  * fSaveTransform and the input branch will be modified to contain AtPadFFTs.
12  *
13  * Adam Anthony 4/20/22
14  */
